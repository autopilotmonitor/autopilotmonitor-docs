## What is Autopilot Monitor?

Autopilot Monitor gives IT admins and MSPs deep insight into the Windows Autopilot enrollment process — tracking device provisioning sessions, the Enrollment Status Page (ESP), app and script installation, and completion in real time, so you can spot and resolve problems before they reach your users.
